1. AI doesn't truly "think" or "understand" - it finds patterns in data
2. The human brain is incredibly efficient (20 watts vs thousands for AI)
3. AI needs millions of examples to learn; humans need just a few
4. Neural networks learn by adjusting "weights" on connections
5. AI lacks common sense because it hasn't experienced the real world
